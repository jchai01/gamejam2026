## Description

Gather resources from distant planets to ensure the survival of The Stellar Pacemaker - humanity's final hope against the dying sun. However, everything isn't as it seems.

A fast-paced 2D shmup/bullet-hell with a sci-fi story! Created for the [Gamedev.js Jam 2026](https://itch.io/jam/gamedevjs-2026), with the theme of "Machines".

---

## CONTROLS

| Key | Action |
| --- | --- |
| **Arrow Keys** | Move |
| **Space** | Shoot |
| **X** | skip to action for some scenes (it will jump only after the current dialogue finishes) |

## Development

Run server:
```
python -m http.server
```

Or (auto reload on file changes):

```
npx browser-sync start --server --files "."
```

Debug keys (press during menu screen):

| Key | Action |
| --- | --- |
| 2 | Mission 1 - return |
| 3 | Mission 2 |
| 4 | Mission 2 - return |
| 5 | Mission 3 |
| 6 | Mission 3 - return |
| 7 | Final mission |
| 8 | Ending sequence |
